# 回文駅名検索

回文になっている駅名を探します。
ローマ字で回文になっている駅名も表示しちゃいました。

データは http://www5a.biglobe.ne.jp/~harako/data/station.htm に載っているものをそのまま使うと超イイ感じになります。

ちなみに元データ2014/5/28版を使うと、回文の駅名は32駅、ローマ字で回文になっている駅名は16駅あります。
両方に含まれるのは「飯井駅」のみです。
